# 餐厅点菜系统

## 项目简介

本系统是一个纯前端实现的餐厅点菜信息系统，包含移动端点餐App和PC端管理后台。代码已按功能拆分为独立的 HTML、CSS、JavaScript 文件，方便在 VSCode 中直接编辑和调试。

- **index.html** — 入口引导页
- **app.html** — 移动端顾客点餐界面
- **admin.html** — PC端商家管理后台
- **css/** — 样式文件目录
- **js/** — 脚本文件目录

## 运行方式

### 方式一：VSCode + Live Server（推荐）

1. 安装 VSCode 插件 **Live Server**（作者：Ritwick Dey）
2. 在 VSCode 中打开本文件夹
3. 右键点击 `index.html` → 选择 **"Open with Live Server"**
4. 浏览器会自动打开，点击按钮选择进入 App 或 Admin

### 方式二：直接双击打开

- 双击 `index.html` 用浏览器打开即可
- 建议用 Chrome 或 Edge 浏览器

## 使用说明

### 演示流程

1. **打开管理后台**（admin.html）
   - 进入「菜品管理」添加几个菜品
   - 进入「桌台管理」添加几个桌位

2. **打开移动端 App**（app.html）
   - 选择桌位进入点餐
   - 添加菜品到购物车，点击「去下单」

3. **回到管理后台**
   - 在「桌台管理」或「订单管理」查看新订单
   - 标记菜品出餐状态
   - 查看数据概览实时更新

## 技术特点

- **纯前端实现**：HTML + CSS + JavaScript，无需后端服务器
- **数据共享**：两端共用 `localStorage` 数据，实时同步
- **会话隔离**：使用 `currentSessionId` 机制，不同批次客人订单互不干扰
- **响应式设计**：App 端适配手机屏幕，Admin 端适配桌面屏幕

## 文件说明

| 文件/目录 | 说明 |
|-----------|------|
| `index.html` | 入口引导页 |
| `app.html` | 移动端点餐App（引用 css/app.css + js/data.js + js/app.js） |
| `admin.html` | PC端管理后台（引用 css/admin.css + js/data.js + js/admin.js） |
| `css/index.css` | 入口页样式 |
| `css/app.css` | 移动端App样式 |
| `css/admin.css` | 管理后台样式 |
| `js/data.js` | 共享数据层：预置数据、localStorage读写、数据迁移工具 |
| `js/app.js` | 移动端业务逻辑：选桌、点餐、购物车、订单、评价 |
| `js/admin.js` | 管理后台业务逻辑：订单/桌台/菜品/分类管理、报表、设置 |

## 注意事项

- `localStorage` 数据存储在浏览器中，换电脑后数据不会同步
- 系统已预置 12 个桌位和 15 道菜品，打开即可直接演示
- 建议使用 Chrome 或 Edge 浏览器，避免兼容性问题
- 如需修改样式，直接编辑 `css/` 目录下的对应文件
- 如需修改业务逻辑，直接编辑 `js/` 目录下的对应文件
